# F1 Visualizer Website

Premium Formula 1-inspired landing page for installer downloads.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Notes

- The download buttons currently point to the release notes section.
- Replace the `href` values in `src/App.tsx` with real installer URLs once packaging is ready.
- Add Express only if you need live release metadata, auth, or download tracking.
